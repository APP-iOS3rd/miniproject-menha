## 🛠️ 개발 환경
- UI: UIKit
- Framework: Firebase(Firestore, Fireabas-authentication Dynamic Links)
- Tool: Xcode, Github, Figma


<br>

## ⚙️ 주요 기능
- 회원가입, 로그인
- 멘토링 일정 등록, 수정, 삭제
- 멘토링 그룹생성(그룹 초대코드)
